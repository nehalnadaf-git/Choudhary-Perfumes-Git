# Choudhary Perfumes - E-Commerce Website

A premium e-commerce platform for "Choudhary Perfumes", featuring luxury attars and designer perfume master copies. Built with Next.js 14, Tailwind CSS, and TypeScript.

## Features

- **Royal Luxury Design**: Custom color palette (Champagne Gold, Deep Black) and typography.
- **Product Catalog**: Filterable and searchable product listing.
- **WhatsApp Integration**: Direct "Buy on WhatsApp" buttons with pre-filled messages.
- **Responsive**: Fully optimized for mobile and desktop.
- **Admin Panel (Mock)**: Frontend-only dashboard for product management demonstration.

## Technologies

- [Next.js 14](https://nextjs.org/) (App Router)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/) (Animations)
- [React Icons](https://react-icons.github.io/react-icons/)
- [TypeScript](https://www.typescriptlang.org/)

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  **Open in browser**:
    Visit [http://localhost:3000](http://localhost:3000)

## Project Structure

- `app/`: Next.js App Router pages (`page.tsx`, `products/page.tsx`, etc.)
- `components/`: Reusable UI components (`Header`, `Hero`, `ProductCard`, etc.)
- `lib/`: Data and types (`products.ts`, `types.ts`)
- `public/`: Static assets (images)

## Deployment

This project is optimized for deployment on Vercel.

1.  Push code to GitHub.
2.  Import project in Vercel.
3.  Deploy!
